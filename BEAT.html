Attendance Tracking System for Enterprise Bath

CONCEPT

Enterprise Bath is a University of Bath department which offers advice, business mentors, competitions, masterclasses, expert talks, placements and networking opportunities to aspiring entrepreneurs and intrapreneurs through an extra-curricular programme of events. This project aimed to create a secure and scalable attendance tracking system to enhance engagement and data accessibility for the entrepreneurial community.

TECHNOLOGIES & FEATURES

The system integrated modern technologies to ensure high efficiency and usability:

  • A GDPR-compliant PostgreSQL database for secure and efficient attendance data management.
  • Automated data collection using LDAP queries and RFID card readers for accurate tracking.
  • A responsive Power BI dashboard for real-time attendance analysis and reporting.
  • Cloud-based technology for scalability and seamless integration.
  • Comprehensive documentation for ease of maintenance and long-term usability.

PROCESS

Development involved:

  • Designing and implementing a PostgreSQL database optimized for performance and data protection.
  • Setting up secure, automated data collection pipelines using LDAP and RFID technologies.
  • Creating intuitive Power BI visualizations tailored to the needs of stakeholders.

OUTCOME

The system successfully streamlined attendance tracking, providing insightful analytics and ensuring data security. It was well-received by the entrepreneurial community for its usability and efficiency.
